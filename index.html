My Projects

Hang Man

Web based game created with Phaser engine and Javascript. This game consist of three main scene , Main Menu scene, Game Play Scene , and Game Over or Win scene. In this game player have to guess the word to save the man and win the game. Player will be given 3 clue per game and if the player guess the wrong letter 7 times , the man will be hanged. This Game constructed with a lots of assets ,a lots of revision and time seacrhing on github :) , need 3 full months to finish this game.

Forward Collision Mitigation System Android

Android based app with Tensorflow Lite engine to detect object (vehicle), in this project the object consist of cars, motorcycle, bus, truck. the detected object will be measured the distance from camera and callculate if the object too close to the camera ,waring will be displayed. this app work like safety fiture on a car called Forward Collision Mitigation (FCM), but the difference is the app can't apply brakes automaticly only visual and audio warn the driver. This app was build with custom model tensorflow, custom model made from thousand of real photos taken in indonesian roads and trained using ssd_mobilenet V2 library.
